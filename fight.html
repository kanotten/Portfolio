Experiences

I am developing a 2D fighting game for two players, primarily using JavaScript and CSS. The game, which features two distinct sprites, is still in progress. At the moment, I am working on creating player boundaries and integrating the sprites. The project involves HTML for structure, JavaScript for game logic, CSS for styling, and sprites for character representation. As a newcomer to game development, this project is a hands-on learning experience. I am exploring new mechanics such as establishing rules, defining character boundaries, and ensuring smooth movements. Although I am nearing the completion of the borders, there is still significant work to be done.
